Git 通过 SSH 上传到 Github

之前我一直都在用Git的https方式部署项目，每次都要输密码，都不懂我是怎么过来的，今天才发现还有SSH方式的，方便多啦!

下载 OPEN SSH

  1. 首先，在终端里输入apt install openssh
    • 等待安装完成
  2. 在终端中输入ssh-keygen -t rsa
    • 等待密匙生成后
  3. 在终端中输入cat ~/.ssh/id_rsa.pub
